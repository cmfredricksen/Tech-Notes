How to Clone a Repository from Github to your Local computer

  • On Github navigate to the repo you want to clone to your computer

  • Click Clone and copy the url posted there

  • Open Git Bash

  • Change Directory (cd) to the one you want to place the repo in

  • Git Init

  • Git Clone --url of remote repo--

  • Git Remote add origin --url of remote repo--

    • Once you have cloned the repo then do the git pull command to bring the files from your remote repo to the desktop.

  • Git Pull --url of remote repo--
